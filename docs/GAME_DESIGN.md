# CuteStation 遊戲設計文件

## 專案概述
- **類型**：2D 橫向平台捲軸動作遊戲
- **風格**：可愛風
- **技術棧**：Tauri V2 + Svelte 5 + TypeScript + PixiJS v8

## 故事概要
騎士奉公主之命追回被盜賊偷走的寶物，途經六大場景後發現寶物是惡魔封印，盜賊實為英雄。最終與惡魔決戰。

## 關卡結構
| 大關 | 場景 | 小關 | Boss |
|------|------|------|------|
| 1 | 白色宮殿 | 3 | ✓ |
| 2 | 森林 | 3 | ✓ |
| 3 | 海洋 | 3 | ✓ |
| 4 | 雪山 | 3 | ✓ |
| 5 | 火山 | 3 | ✓ |
| 6 | 惡魔據點 | 3 | ✓ 最終Boss |

每小關含 5 個收集幣

## 可選角色
- 騎士
- 公主
- （操作方式相同，外觀不同）

## 核心機制

### 移動系統
- 左右移動
- 奔跑（按住特定鍵加速）

### 跳躍系統
- 基本跳躍

### 攻擊系統
- 近戰攻擊
- Homing Attack：空中時鎖定範圍內敵人，衝撞攻擊

### 收集系統
- 每關 5 個收集幣

## 輸入支援
- 鍵盤（PC）
- 觸控虛擬按鍵（手機）
- 手把控制器
